Our Story

Justification 1: Color

Since our topic is about technology and how it will change our world, we tried to find a color palette that suits the topic. In our research, we found that that that most tech companies use blue in much higher rate in their logo and designs, followed by white, black and red. Additionally we found that they use blue because it invokes stability and security and it represents intelligence and trustworthiness, which makes them instantly establish credibility and authority with the client. So we tried to make our color palette a blue based one, with one dark and two light colors.

Justification 2: Layout and structure

For our design and layout, we thought to use a minimal website design such as float design with each section/paragraph is considered as a standalone section that it is in a div in its on. The main advantage of using this type of design is the flexibility and ease of using the website content regardless of the screen size.Additionally, we tried to rely on repeating items or repeating groups. Besides that, they are followed in modern designs. The consistency they give, gives the user kind of a roadmap on how to use the website and navigate easily.

Justification 3: Code: Flex

Flex (flexibility), one of relatively newest techniques in CSS. It basically depends on flexibility and availability of spaces when setting the height and widths of an element. Additionally, it is very useful for responsive websites designs because it depends on the availability, which in future, can help this website to be responsive. Finally, it allows us to order and alter the items easily. Because of this we thought it would be a great choice for our website.

Justification 4: Code: JavaScript

JavaScript is the best way to program any website. By using the Local Storage function in JavaScript we are able to display the user’s first name and last name as soon as he clicks on the Register button. In addition to that we have used JavaScript in the slideshow you have seen above containing all the mock-up images we have created for the website. JavaScript has allowed us to make our website more user-friendly and without a doubt this will encourage the user to re-use it.
